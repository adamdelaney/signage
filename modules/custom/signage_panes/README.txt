-- SUMMARY --

The Signage Panes module provides a Ctools content type for various panes in
signage.


-- REQUIREMENTS --

* Ctools - http://drupal.org/project/ctools


-- INSTALLATION --

* Install as usual, see
  http://drupal.org/documentation/install/modules-themes/modules-7


-- USAGE --

* If using Panels, when adding content to a panel, new panes are available.
